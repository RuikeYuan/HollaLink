# 中国创业者赴荷开店的身份/签证路径

在荷兰合法经营门店，前提是持有允许"自雇/创业"的合法居留身份。常见路径：

## 1. Zelfstandig ondernemer（自雇创业者）居留
- 面向非欧盟公民独立创业，IND（荷兰移民局）会用"积分制"评估商业计划（个人经验、商业计划本身、对荷兰的经济附加值三个维度打分）。
- 需要提交详细商业计划书（这正是本平台"报告生成"功能的核心输出场景之一）。
- 审批周期通常 3-6 个月，期间不能提前开业运营依赖此身份。

## 2. Startup 签证（一年期）
- 面向有创新性商业模式（非传统零售/餐饮）的创业者，需要一名 IND 认证的"辅导机构"（facilitator）提供一年辅导。
- 传统开店（如餐馆、零售）通常不满足"创新性"要求，更适合走"自雇创业者"或投资类路径。

## 3. 投资者居留（Investor / DAFT 类专用通道）
- 中国大陆公民不适用中荷友好通商航海条约（DAFT，该条约仅适用于美国、日本等特定国家公民），常见误传"中国人可以用美国式投资签证"是错误信息，需要在咨询中明确澄清。

## 4. 配偶/家庭团聚身份
- 若创业者本人已通过配偶居留在荷兰合法居留且该居留类型允许自由工作，则可以直接以个人身份注册 Eenmanszaak/BV 经营，无需单独申请创业居留，这是很多咨询案例中最容易被忽略的"更简单路径"。

## 5. BV + 董事身份（较常见的实操路径之一）
- 部分创业者选择先设立 BV，自己作为董事，但"董事身份"本身不自动等于合法居留/工作许可，仍需要单独满足居留条件（如已持其他类型居留，或走自雇创业者评估路径），不能想当然认为"开公司就能拿身份"。

## 给顾问 Agent 的强制提醒
只要用户提到"我还没有荷兰身份"或"用旅游签证考察"，必须明确提醒：
- 旅游签证只能考察市场、不能实际经营或长期雇佣自己在店内工作。
- 身份路径和商业计划是强相关的两件事，建议尽早同步咨询移民律师（IND 认证顾问）与本平台的商业顾问服务，避免"店开好了但身份卡住"的常见失败模式。
