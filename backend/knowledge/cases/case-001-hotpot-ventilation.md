# 案例 001：火锅店排烟系统成本低估

- **行业**：火锅店
- **城市**：（未公开具体城市，同类问题在阿姆斯特丹/鹿特丹均有发生）
- **投资规模**：约 €200,000
- **结果**：开业后被市政环境部门要求整改，最终因追加排烟设备投入和停业整改期损失，项目亏损退出。

## 发生了什么
创业者签订商铺长期租约并完成大部分室内装修后，才发现：
1. 该商铺此前经营业态是普通简餐，原有排烟系统达不到火锅/中式爆炒高油烟负荷的《活动设施法令》标准。
2. 房东在签约时未主动告知排烟系统的实际规格上限，创业者也未在签约前委托专业机构检测评估。
3. 补装静电油烟净化器（ESP）+ 排烟管道改造，追加成本约 €30,000-40,000，远超原始预算预留（原预算仅预留 €5,000 装修余量）。
4. 市政要求整改期间门店停业，产生了额外的租金和人工"沉没成本"。

## 根本原因
- 未核实 [[zoning-bestemmingsplan]] 和该地址此前的排烟系统真实规格。
- 未在租约中加入"以获得必要环境许可为生效条件"的保护性条款。
- 商业计划阶段的成本预算未参考同类业态（高油烟餐饮）的真实装修成本区间，仅参考了普通餐饮的经验数字。

## 经验总结（供 Agent 引用）
1. **签合同前必须书面确认商业用途和现有设施规格是否满足目标业态**，尤其是高油烟餐饮（火锅、烧烤、中式爆炒）。
2. 装修预算中，"隐藏工程"（排烟、排污、消防、无障碍改造）应预留至少总装修预算的 20-30% 作为缓冲，而不是只按"看得见"的软装/门头去估算。
3. 建议顾问在成本计算器输出中，对火锅/烧烤类业态默认标注"排烟系统专项评估"为高优先级风险项，并给出更高的装修成本区间上限。
