# 荷兰本地服务商参考名录（律师 / 会计 / 商铺中介）

> **重要说明（供顾问 Agent 严格遵守）**：以下机构是通过公开信息检索到的、**目前真实存在、可自行核实联系的机构**，不是"荷兰开店通"平台已经签约认证的合作伙伴，也没有经过本平台对其服务质量的实地验证。回答用户时必须清楚说明这一点——"以下是市面上可查到的选项，建议自行联系核实资质、报价和口碑，再决定是否合作"，不能说"我们推荐/保证这家机构"，避免造成平台为第三方服务背书的误解。

## 律师事务所（移民法 / 商业法，有中文服务）
- **Everaert Advocaten（移福嘉律师事务所）** — 设有 China Desk，自 1982 年成立，专门服务希望经由荷兰进入欧洲市场的中国企业，处理居留许可、劳动许可、公司设立咨询（everaert.nl）
- **Jing Law Firm（景舜律师事务所）** — 创始人郭景是荷兰本土执业的华人律师，处理商事诉讼、并购投资类业务（jinglawfirm.nl）
- **HULX（宏丹律师事务所，海牙）** — 设有中文部（hulx.nl）
- **Pieters Advocaten（柏德律师事务所，乌得勒支）** — 设有中国事务部，处理移民法、劳动法（pietersadvocaten.cn）
- **Q Advocaten（鹿特丹）** — 提供中文页面（qadvocaten.nl）

## 会计 / 税务服务商
- **Neplus Financial Service（荷嘉会计师事务所）** — 明确服务在荷华人企业与驻荷中资公司，提供公司注册、税号/银行账户申请、VAT 申报、跨境电商 VAT 注册等服务（neplusfs.com）

## 官方/系统性查找渠道（比零散搜索更全面）
- **NOvA（荷兰律师协会）官方查询系统** — 可按专业方向（如 vreemdelingenrecht 移民法、ondernemingsrecht 商业法）筛选全荷兰注册律师，覆盖面远大于上面几家
- **NBA（荷兰注册会计师协会）/ RB（税务顾问协会）官方注册库** — 同理，比零散搜索更系统
- **IND 官方认证 facilitator 名单** — 如果客户计划走 Startup Visa 路径，IND 官网有一份官方认证辅导机构名单，这些机构本身就是做"辅导外国创业者"这件事的

## 商铺中介 / Horeca 专业地产经纪
餐饮/零售选址建议优先找**horeca 专项中介**而非普通住宅地产经纪，专项中介更懂用途核实、盘店（bedrijfsovername）等行业惯例：
- **Adhoc Horecamakelaars** — 阿姆斯特丹地区 horeca 商铺租赁/买卖专项中介（adhoc-horecamakelaars.nl）
- **Van Alkemare Horecamakelaardij** — 覆盖北荷兰、南荷兰、乌得勒支、弗莱福兰多省的 horeca 买卖/租赁/估值（vanalkemare.nl）
- **Dam Horecamakelaar / Klaassen Horecamakelaars** — 阿姆斯特丹地区 horeca 商铺中介
- **Horecasite.nl / Bedrijfspand.com** — 全国性 horeca 商铺/生意转让挂牌平台，超过 2,200 个 horeca 生意在售/转让，找"接手转让店"（见 [[cases/case-001-hotpot-ventilation.md|华人餐饮真实现状]]）优先从这类平台筛选

## 给顾问 Agent 的使用建议
1. 当用户问"去哪找律师/会计/商铺"这类问题时，可以提及以上名录中相关的 1-2 家作为**参考起点**，但必须明确这是"公开可查的选项"而不是"平台认证合作方"。
2. 涉及找店/接手转让店的场景，优先提到 horeca 专项中介和 Horecasite.nl 这类专项平台，而不是笼统建议"找地产中介"。
3. 提醒用户：无论选择哪家机构，正式签约或付费前都应该自行核实资质（律师可在 NOvA 官网核实执业登记，会计可在 NBA 官网核实注册状态）。
4. 如果本平台后续与某家机构达成正式合作关系，应更新此文档明确标注"平台认证合作伙伴"，与本列表的其余"仅供参考"条目做区分。
